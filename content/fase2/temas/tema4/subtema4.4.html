4.4 La declaración de la clase principal

La declaración de la clase es la estructura fundamental donde reside todo el código que escribimos en Java. A diferencia de otros lenguajes donde el código puede existir "suelto", en Java absolutamente todo debe estar contenido dentro de una clase. La clase principal de un programa es aquella que contiene el método main, el punto de partida donde la JVM comienza a ejecutar nuestro código. Comprender cómo declarar correctamente una clase, qué significa cada parte de la declaración, y qué reglas debe seguir es esencial para escribir programas Java válidos y bien estructurados.

Anatomía de la declaración de clase

Forma básica:

class NombreClase {
    // Aquí va el contenido de la clase
}

Componentes:

  1. class - Palabra clave que indica "aquí comienza una clase"
  2. NombreClase - El nombre que le damos a la clase
  3. { } - Las llaves que encierran todo el contenido de la clase

Ejemplo mínimo de declaración de clase

class MiPrimeraClase {
    public static void main(String[] args) {
        System.out.println("Hola desde mi primera clase");
    }
}

Desglose:

class MiPrimeraClase {
  • class = palabra clave obligatoria
  • MiPrimeraClase = nombre de la clase
  • { = abre el contenido de la clase
public static void main(String[] args) {
    System.out.println("Hola desde mi primera clase");
}

Método main (punto de entrada del programa)

}

} = cierra el contenido de la clase

Reglas para el nombre de la clase

Reglas obligatorias (técnicas):

  1. Debe comenzar con letra (A-Z, a-z)
  2. Puede contener letras, números y guión bajo (_)
  3. NO puede contener espacios
  4. NO puede ser una palabra reservada de Java (class, int, public, etc.)

Ejemplos válidos técnicamente:

class Calculadora { }   // ✓ Válido
class Calculadora2 { }  // ✓ Válido (con número)
class Mi_Clase { }      // ✓ Válido (con guión bajo)
class calculadora { }   // ✓ Válido (minúsculas)

Ejemplos inválidos:

class 2Calculadora { }       // ✗ No puede empezar con número
class Mi Clase { }           // ✗ No puede tener espacios
class class { }              // ✗ "class" es palabra reservada
class Calculadora-Básica { } // ✗ No puede tener guión (-)

Convención de nombres Java (PascalCase)

Aunque técnicamente son válidos varios estilos, Java tiene una convención estándar que todos los programadores Java profesionales siguen:

PascalCase: Primera letra de cada palabra en mayúscula, sin espacios

// ✓ RECOMENDADO (estilo Java estándar)
class CalculadoraDescuento { }
class SistemaInventario { }
class ReporteVentasMensuales { }
class ControlEmpleados { }
// ✗ NO RECOMENDADO (funcionan pero no son estilo Java)
class calculadora_descuento { } // Estilo snake_case (Python)
class calculadoradescuento { }  // Todo junto, difícil de leer
class CALCULADORADESCUENTO { }  // Todo mayúsculas

¿Por qué seguir la convención?

  • Todo código Java profesional la sigue
  • Hace el código más legible
  • Otros programadores esperan este estilo
  • IDEs y herramientas asumen este estilo

La palabra clave public

Hasta ahora hemos visto:

class MiClase {
    // ...
}

Pero frecuentemente verá:

public class MiClase {
    // ...
}

¿Qué significa public?

public significa que la clase es visible desde cualquier parte.

Por ahora, la regla simple:

Para programas ejecutables (con método main), siempre use public:

public class MiPrograma {
    public static void main(String[] args) {
        // Código del programa
    }
}

Razón: El programa principal debe ser accesible para que la JVM pueda ejecutarlo.

Estructura completa de una clase ejecutable

public class NombrePrograma {
    public static void main(String[] args) {
        // Aquí van las instrucciones del programa
    }
}

Esta es la estructura que usaremos para TODOS nuestros programas por ahora.

Ejemplo: Calculadora de precio con descuento

Archivo: CalculadoraPrecio.java

public class CalculadoraPrecio {
    public static void main(String[] args) {
        // Datos del producto
        double precioOriginal = 850.00;
        double porcentajeDescuento = 20.00;

        // Calcular descuento
        double montoDescuento = precioOriginal * porcentajeDescuento / 100;

        // Calcular precio final
        double precioFinal = precioOriginal - montoDescuento;

        // Mostrar resultados
        System.out.println("===== CALCULADORA DE PRECIO =====");
        System.out.println("Precio original: Lps " + precioOriginal);
        System.out.println("Descuento (" + porcentajeDescuento + "%): Lps " + montoDescuento);
        System.out.println("Precio final: Lps " + precioFinal);
        System.out.println("=================================");
    }
}

Declaración de la clase:

public class CalculadoraPrecio {
  • public = clase visible públicamente
  • class = declaración de clase
  • CalculadoraPrecio = nombre de la clase
  • { = inicio del contenido

Salida:

===== CALCULADORA DE PRECIO =====
Precio original: Lps 850.0
Descuento (20.0%): Lps 170.0
Precio final: Lps 680.0
=================================

Contenido de la clase: El método main

Dentro de las llaves { } de la clase va el contenido. Para programas ejecutables, el contenido principal es el método main.

public class MiPrograma {

    public static void main(String[] args) {
        // Instrucciones del programa
    }

}

El método main debe tener exactamente esta firma:

public static void main(String[] args)
  • public = accesible públicamente
  • static = pertenece a la clase (no necesita crear objeto)
  • void = no retorna valor
  • main = nombre del método (debe ser exactamente "main")
  • String[] args = parámetro que recibe argumentos

Por ahora, memorice esta línea exactamente como está. La explicaremos en detalle en el siguiente subtema.

Ejemplo: Sistema de cálculo de impuesto

Archivo: CalculadoraImpuesto.java

public class CalculadoraImpuesto {
    public static void main(String[] args) {
        // Datos de la venta
        double montoVenta = 5000.00;
        double tasaImpuesto = 15.00;

        // Calcular impuesto
        double montoImpuesto = montoVenta * tasaImpuesto / 100;

        // Calcular total
        double montoTotal = montoVenta + montoImpuesto;

        // Mostrar desglose
        System.out.println("===== CÁLCULO DE IMPUESTO =====");
        System.out.println("Monto de venta: Lps " + montoVenta);
        System.out.println("Impuesto (" + tasaImpuesto + "%): Lps " + montoImpuesto);
        System.out.println("-------------------------------");
        System.out.println("Total a pagar: Lps " + montoTotal);
        System.out.println("===============================");
    }
}

Estructura de la clase:

public class CalculadoraImpuesto { // Declaración de la clase

    public static void main(String[] args) { // Método main
        // Código del programa
    }

} // Fin de la clase

Las llaves { } delimitan la clase

Todo lo que está entre { y } pertenece a la clase:

public class Ejemplo { // ← Abre la clase

    public static void main(String[] args) {
        System.out.println("Dentro de la clase");
    }

} // ← Cierra la clase

Fuera de las llaves = fuera de la clase = ERROR

public class Ejemplo {
    public static void main(String[] args) {
        System.out.println("Correcto");
    }
}

System.out.println("Esto está fuera de la clase"); // ✗ ERROR

Error de compilación:

error: class, interface, or enum expected

En Java, TODO debe estar dentro de una clase.

Ejemplo: Calculadora de salario neto

Archivo: CalculadoraSalarioNeto.java

public class CalculadoraSalarioNeto {
    public static void main(String[] args) {
        // Salario bruto mensual
        double salarioBruto = 15000.00;

        // Deducciones
        double deduccionSeguroSocial = salarioBruto * 3.5 / 100;
        double deduccionImpuestoRenta = salarioBruto * 10.0 / 100;

        // Calcular total de deducciones
        double totalDeducciones = deduccionSeguroSocial + deduccionImpuestoRenta;

        // Calcular salario neto
        double salarioNeto = salarioBruto - totalDeducciones;

        // Mostrar desglose
        System.out.println("===== DESGLOSE DE SALARIO =====");
        System.out.println("Salario bruto: Lps " + salarioBruto);
        System.out.println("-------------------------------");
        System.out.println("Deducciones:");
        System.out.println("  Seguro Social: Lps " + deduccionSeguroSocial);
        System.out.println("  Imp. Renta: Lps " + deduccionImpuestoRenta);
        System.out.println("  Total: Lps " + totalDeducciones);
        System.out.println("-------------------------------");
        System.out.println("Salario neto: Lps " + salarioNeto);
        System.out.println("===============================");
    }
}

Salida:

-------------------------------

  Seguro Social: Lps 525.0

Errores comunes en la declaración de clase

Error 1: Olvidar la palabra class

public MiPrograma { // ✗ Falta "class"
    public static void main(String[] args) {
        // ...
    }
}

Solución:

public class MiPrograma { // ✓ Correcto
    // ...
}

Error 2: Nombre de clase con espacios

public class Mi Programa { // ✗ Espacio en el nombre
    // ...
}

Solución:

public class MiPrograma { // ✓ Sin espacios
    // ...
}

Error 3: Olvidar llaves

public class MiPrograma // ✗ Falta {
    public static void main(String[] args) {
        System.out.println("Error");
    }
// ✗ Falta }

Solución:

public class MiPrograma { // ✓ Con llave de apertura
    public static void main(String[] args) {
        System.out.println("Correcto");
    }
} // ✓ Con llave de cierre

Error 4: Llaves desbalanceadas

public class MiPrograma {
    public static void main(String[] args) {
        System.out.println("Error");
    }
// ✗ Falta cerrar llave de la clase

Solución: Contar las llaves:

  • Cada { debe tener su correspondiente }

Buenas prácticas en la declaración de clase

1. Siempre usar public para la clase principal

public class ProgramaPrincipal { // ✓ Con public
    public static void main(String[] args) {
        // ...
    }
}

2. Nombre descriptivo en PascalCase

// ✓ Descriptivo
public class CalculadoraImpuestoVentas {
    // ...
}

// ✗ Genérico
public class Programa1 {
    // ...
}

3. Una clase por archivo

proyecto/
├── CalculadoraDescuento.java → Una clase
├── ReporteVentas.java        → Una clase
└── SistemaNomina.java        → Una clase

4. Indentar correctamente el contenido

public class Ejemplo {
    public static void main(String[] args) { // Indentado 4 espacios
        System.out.println("Hola");          // Indentado 8 espacios
    }
}

La indentación (sangría) hace el código más legible.

Plantilla para nuevos programas

Use esta plantilla para crear nuevos programas:

public class NombreDelPrograma {
    public static void main(String[] args) {
        // Su código aquí
    }
}

Reemplace:

  • NombreDelPrograma con el nombre de su programa (sin espacios, PascalCase)
  • Nombre del archivo debe coincidir: NombreDelPrograma.java

Ejemplo final: Control de gastos mensuales

Archivo: ControlGastosMensuales.java

public class ControlGastosMensuales {
    public static void main(String[] args) {
        // Gastos del mes
        double gastoAlquiler = 5000.00;
        double gastoAlimentacion = 3500.00;
        double gastoTransporte = 1200.00;
        double gastoServicios = 2500.00;
        double gastosVarios = 1800.00;

        // Calcular total de gastos
        double totalGastos = gastoAlquiler + gastoAlimentacion +
                             gastoTransporte + gastoServicios + gastosVarios;

        // Ingreso mensual
        double ingresoMensual = 18000.00;

        // Calcular saldo
        double saldoMensual = ingresoMensual - totalGastos;

        // Mostrar reporte
        System.out.println("===== CONTROL DE GASTOS MENSUALES =====");
        System.out.println("Ingresos: Lps " + ingresoMensual);
        System.out.println("---------------------------------------");
        System.out.println("Gastos:");
        System.out.println("  Alquiler: Lps " + gastoAlquiler);
        System.out.println("  Alimentación: Lps " + gastoAlimentacion);
        System.out.println("  Transporte: Lps " + gastoTransporte);
        System.out.println("  Servicios: Lps " + gastoServicios);
        System.out.println("  Varios: Lps " + gastosVarios);
        System.out.println("  ----------------");
        System.out.println("  Total gastos: Lps " + totalGastos);
        System.out.println("---------------------------------------");
        System.out.println("Saldo mensual: Lps " + saldoMensual);
        System.out.println("=======================================");
    }
}

Declaración de la clase:

  • ✓ Usa public class
  • ✓ Nombre descriptivo (ControlGastosMensuales)
  • ✓ Sigue PascalCase
  • ✓ Coincide con nombre de archivo
  • ✓ Llaves balanceadas
  • ✓ Contiene método main

La declaración de la clase es la estructura fundamental que contiene todo nuestro código Java. Siguiendo las reglas simples de usar public class, nombres descriptivos en PascalCase, y asegurando que el nombre coincide con el archivo, creamos programas bien estructurados que compilan sin errores y son fáciles de entender y mantener.

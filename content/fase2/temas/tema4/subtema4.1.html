4.1 Qué es un programa Java desde una perspectiva técnica

Un programa Java es el conjunto de instrucciones que escribimos en archivos de texto con extensión .java, que luego son transformadas por el compilador en un formato especial que la máquina virtual de Java puede ejecutar. Comprender qué es técnicamente un programa Java, cómo está organizado, qué reglas fundamentales debe seguir, y qué sucede desde que escribimos el código hasta que se ejecuta, es esencial para comenzar a escribir programas correctos. Este subtema establece la base conceptual sobre la cual se construyen todos los programas Java que desarrollaremos en este plan maestro.

Definición técnica básica

Un programa Java consiste en:

  1. Uno o más archivos de texto con extensión .java
  2. Código escrito siguiendo las reglas del lenguaje Java
  3. Que al ser compilados producen archivos .class con bytecode
  4. Que la JVM puede ejecutar para realizar las tareas que programamos

Analogía simple:

Piense en un programa Java como una receta de cocina:

  • Archivo .java = La receta escrita en español (legible por humanos)
  • Compilación = Traducir la receta a un idioma especial que entiende la cocina
  • Archivo .class = La receta traducida (bytecode)
  • JVM = La cocina que ejecuta la receta traducida
  • Resultado = El plato preparado (programa ejecutándose)

El programa Java más simple posible

El programa Java más pequeño que puede compilar y ejecutar es:

class Minimo {
    public static void main(String[] args) {
    }
}

Análisis de cada parte:

class Minimo

  • class es una palabra clave que indica "aquí comienza una clase"
  • Minimo es el nombre que le damos a esta clase
  • En Java, TODO el código debe estar dentro de una clase

{ y }

  • Las llaves delimitan "el contenido" de la clase
  • Todo lo que esté entre { y } pertenece a la clase

public static void main(String[] args)

  • Esta es la "firma" del método main
  • Es el punto de entrada del programa (donde comienza a ejecutarse)
  • Por ahora, solo necesita saber que TODO programa Java ejecutable debe tener esta línea exacta

El método main está vacío { }

  • No hace nada, pero es técnicamente válido
  • Se puede compilar y ejecutar sin errores

Un programa que realmente hace algo

class PrimerPrograma {
    public static void main(String[] args) {
        System.out.println("Mi primer programa Java");
    }
}

Lo nuevo aquí:

System.out.println("Mi primer programa Java");

  • System.out.println es un comando para mostrar texto en pantalla
  • El texto entre comillas "..." es lo que se mostrará
  • El ; al final indica que la instrucción terminó

Guardar, compilar y ejecutar:

Paso 1: Guardar el archivo

  • Nombre del archivo: PrimerPrograma.java (debe coincidir con el nombre después de class)
  • Ubicación: Cualquier carpeta de trabajo

Paso 2: Compilar

javac PrimerPrograma.java

Resultado: Se crea un archivo PrimerPrograma.class

Paso 3: Ejecutar

java PrimerPrograma

Salida en pantalla:

Mi primer programa Java

Componentes fundamentales de un programa Java

Todo programa Java tiene estos elementos básicos:

1. Declaración de clase

class CalculadoraBasica {
    // Aquí va el código
}
  • Toda instrucción ejecutable debe estar dentro de una clase
  • El nombre de la clase debe coincidir con el nombre del archivo
  • Por convención, los nombres de clases empiezan con mayúscula

2. Método main (punto de entrada)

public static void main(String[] args) {
    // Aquí va el código que se ejecuta
}
  • Este método DEBE existir en el programa principal
  • La JVM busca este método para iniciar la ejecución
  • La firma debe ser exactamente como se muestra (por ahora, memorícela tal cual)

3. Sentencias (instrucciones)

System.out.println("Texto a mostrar");
double precioProducto = 150.00;
double descuento = 20.00;
double precioFinal = precioProducto - descuento;
  • Cada instrucción termina con ; (punto y coma)
  • Las instrucciones se ejecutan en orden de arriba hacia abajo
  • Pueden declarar variables, hacer cálculos, mostrar resultados, etc.

4. Bloques de código { }

class MiClase { // Bloque de la clase
    public static void main(String[] args) { // Bloque del método main
        // Instrucciones aquí
    }
}
  • Las llaves { y } agrupan código
  • Todo lo que está dentro de { y } pertenece al mismo bloque

Ejemplo completo: Programa de cálculo simple

class CalculadoraDescuento {
    public static void main(String[] args) {
        // Declarar variables con valores iniciales
        double precioOriginal = 500.00;
        double porcentajeDescuento = 15.00;

        // Calcular el monto del descuento
        double montoDescuento = precioOriginal * porcentajeDescuento / 100;

        // Calcular el precio final
        double precioFinal = precioOriginal - montoDescuento;

        // Mostrar resultados
        System.out.println("=== Calculadora de Descuentos ===");
        System.out.println("Precio original: Lps " + precioOriginal);
        System.out.println("Descuento: " + porcentajeDescuento + "%");
        System.out.println("Monto descontado: Lps " + montoDescuento);
        System.out.println("Precio final: Lps " + precioFinal);
    }
}

Guardar como: CalculadoraDescuento.java

Compilar:

javac CalculadoraDescuento.java

Ejecutar:

java CalculadoraDescuento

Salida:

=== Calculadora de Descuentos ===
Precio original: Lps 500.0
Descuento: 15.0%
Monto descontado: Lps 75.0
Precio final: Lps 425.0

Ejemplo con cálculo de factura

class CalculadoraFactura {
    public static void main(String[] args) {
        // Datos de la factura
        double subtotal = 1200.00;
        double tasaImpuesto = 15.00;

        // Calcular impuesto
        double montoImpuesto = subtotal * tasaImpuesto / 100;

        // Calcular total
        double totalFactura = subtotal + montoImpuesto;

        // Mostrar factura
        System.out.println("========== FACTURA ==========");
        System.out.println("Subtotal: Lps " + subtotal);
        System.out.println("Impuesto (" + tasaImpuesto + "%): Lps " + montoImpuesto);
        System.out.println("-----------------------------");
        System.out.println("TOTAL: Lps " + totalFactura);
        System.out.println("=============================");
    }
}

Reglas básicas que debe seguir un programa Java

Regla 1: El nombre del archivo debe coincidir con el nombre de la clase

// Archivo: Calculadora.java
class Calculadora {
    // ✓ Correcto: nombre coincide
}
// Archivo: Programa.java
class Calculadora {
    // ✗ Error: nombres no coinciden
}

Regla 2: Todo programa ejecutable debe tener método main

class MiPrograma {
    public static void main(String[] args) {
        // ✓ Correcto: tiene método main
    }
}
class MiPrograma {
    // ✗ Error: sin método main, no se puede ejecutar
}

Regla 3: Las instrucciones deben terminar con punto y coma

System.out.println("Hola"); // ✓ Correcto
System.out.println("Hola") // ✗ Error: falta ;

Regla 4: Mayúsculas y minúsculas importan

class MiPrograma { // ✓ Correcto
}

class miprograma { // ✗ Diferente (no coincidiría con MiPrograma.java)
}

Regla 5: Las llaves deben estar balanceadas

class Ejemplo {
    public static void main(String[] args) {
        System.out.println("Funciona");
    }
}
// ✓ Correcto: 2 llaves abren {, 2 llaves cierran }
class Ejemplo {

        System.out.println("Error");

// ✗ Error: falta cerrar llaves

Características fundamentales de programas Java

1. Todo está en clases

A diferencia de otros lenguajes, en Java no puede haber código "suelto". TODO debe estar dentro de una clase.

2. Compilación obligatoria

No puede ejecutar directamente el archivo .java. Primero debe compilarlo a .class.

3. Portable

El mismo archivo .class funciona en Windows, macOS y Linux sin cambios.

4. Seguro

El compilador verifica muchos errores antes de ejecutar (tipos de datos, sintaxis, etc.)

Ejemplo: Calculadora de inventario

class CalculadoraInventario {
    public static void main(String[] args) {
        // Datos del inventario
        int cantidadProductoA = 50;
        int cantidadProductoB = 30;
        int cantidadProductoC = 75;

        // Calcular total de productos
        int totalProductos = cantidadProductoA + cantidadProductoB + cantidadProductoC;

        // Precios unitarios
        double precioProductoA = 25.50;
        double precioProductoB = 48.00;
        double precioProductoC = 15.75;

        // Calcular valor del inventario
        double valorProductoA = cantidadProductoA * precioProductoA;
        double valorProductoB = cantidadProductoB * precioProductoB;
        double valorProductoC = cantidadProductoC * precioProductoC;

        double valorTotalInventario = valorProductoA + valorProductoB + valorProductoC;

        // Mostrar resultados
        System.out.println("=== REPORTE DE INVENTARIO ===");
        System.out.println("Producto A: " + cantidadProductoA + " unidades - Valor: Lps " + valorProductoA);
        System.out.println("Producto B: " + cantidadProductoB + " unidades - Valor: Lps " + valorProductoB);
        System.out.println("Producto C: " + cantidadProductoC + " unidades - Valor: Lps " + valorProductoC);
        System.out.println("-----------------------------");
        System.out.println("Total de productos: " + totalProductos + " unidades");
        System.out.println("Valor total del inventario: Lps " + valorTotalInventario);
    }
}

Salida:

=== REPORTE DE INVENTARIO ===
Producto A: 50 unidades - Valor: Lps 1275.0
Producto B: 30 unidades - Valor: Lps 1440.0
Producto C: 75 unidades - Valor: Lps 1181.25
-----------------------------
Total de productos: 155 unidades
Valor total del inventario: Lps 3896.25

Un programa Java, desde una perspectiva técnica, es código organizado en clases que sigue reglas estrictas de sintaxis, que debe compilarse antes de ejecutarse, y que la JVM transforma en instrucciones que la computadora puede ejecutar. Aunque al principio puede parecer que hay muchas reglas que recordar, pronto se volverán naturales a medida que practique escribiendo programas simples como los mostrados en este subtema.

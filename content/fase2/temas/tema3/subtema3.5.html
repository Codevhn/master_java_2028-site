3.5: Descarga e instalación del JDK

La descarga e instalación del JDK es el primer paso práctico para establecer un entorno de desarrollo Java funcional. Aunque el proceso es relativamente directo en sistemas operativos modernos, hacerlo correctamente desde el inicio previene problemas posteriores y establece una base sólida para el trabajo de desarrollo. Este subtema cubre el proceso completo desde identificar qué descargar hasta verificar que la instalación fue exitosa.

Paso 1: Seleccionar la distribución y versión

Basándose en los criterios discutidos en el subtema anterior, se recomienda Eclipse Temurin (anteriormente AdoptOpenJDK) como opción general por ser gratuito, open source, bien mantenido con soporte LTS, neutral (no atado a proveedor cloud específico), y ampliamente usado.

Elegir versión:

  • Para proyectos nuevos: Java 21 LTS (versión LTS más reciente)
  • Para proyectos existentes: la versión que el proyecto requiere
  • Para aprendizaje: cualquier versión LTS reciente (17 o 21)

Paso 2: Descargar el JDK

El sitio oficial de Eclipse Temurin es adoptium.net. El proceso de descarga es el siguiente:

  1. Abrir el sitio en el navegador.
  2. Seleccionar opciones:
    • Operating System: Detectado automáticamente (Windows/macOS/Linux)
    • Architecture: Típicamente x64 (o ARM64 para Apple Silicon)
    • Package Type: JDK (no JRE)
    • Version: Seleccionar versión LTS deseada (21, 17, 11, o 8)
  3. Elegir formato de instalador:
    • Windows: .msi (recomendado) o .zip (extracción manual)
    • macOS: .pkg (recomendado) o .tar.gz (extracción manual)
    • Linux: .tar.gz, .deb, o .rpm
  4. Hacer click en el botón de descarga.

Tamaños típicos de descarga:

  • JDK 21: ~180-200 MB
  • JDK 17: ~170-190 MB
  • JDK 11: ~160-180 MB

Paso 3: Instalación en Windows

Usando instalador .msi (recomendado):

  1. Doble click en el archivo .msi descargado. Si aparece UAC, click "Sí" para permitir.
  2. Seguir el wizard:
    • Destination folder: Por defecto C:\Program Files\Eclipse Adoptium\jdk-21.x.x\ (se puede cambiar)
    • Features: Dejar todas las opciones seleccionadas
    • Set JAVA_HOME: Marcar la opción si está disponible
  3. Click "Install" y esperar a que finalice. Click "Finish".

Usando archivo .zip (manual):

  1. Click derecho en el .zip → "Extract All"
  2. Elegir ubicación (ejemplo: C:\Java\jdk-21) y extraer.
  3. Continuar con configuración de variables de entorno (subtema 3.6).

Paso 3: Instalación en macOS

Usando instalador .pkg (recomendado):

  1. Doble click en el archivo .pkg descargado. Si es necesario, aprobar desde "System Preferences → Security & Privacy".
  2. Seguir el wizard: Introduction → License (Agree) → Install (ingresar password de administrador).
  3. Click "Close" al finalizar.

El JDK quedará instalado en:

  /Library/Java/JavaVirtualMachines/temurin-21.jdk/

Usando archivo .tar.gz (manual):

  # Navegar a directorio de descarga
cd ~/Downloads

# Extraer
tar -xzf OpenJDK21U-jdk_x64_mac_hotspot_21.x.x.tar.gz

# Mover a ubicación apropiada
sudo mv jdk-21.x.x.jdk /Library/Java/JavaVirtualMachines/

Paso 3: Instalación en Linux

Método 1: Extracción manual (funciona en todas las distribuciones):

  # Extraer
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.x.x.tar.gz

# Mover a directorio estándar (requiere sudo)
sudo mv jdk-21.x.x /opt/jdk-21

# O instalar en home directory (sin sudo)
mkdir -p ~/java
mv jdk-21.x.x ~/java/jdk-21

Método 2: Paquete .deb (Ubuntu/Debian):

  # Instalar el .deb descargado
sudo dpkg -i OpenJDK21U-jdk_x64_linux_hotspot_21.x.x.deb

# Resolver dependencias si es necesario
sudo apt-get install -f

Método 3: Paquete .rpm (Fedora/RHEL/CentOS):

  # Con rpm
sudo rpm -ivh OpenJDK21U-jdk_x64_linux_hotspot_21.x.x.rpm

# O con yum/dnf
sudo yum install OpenJDK21U-jdk_x64_linux_hotspot_21.x.x.rpm

Método 4: Gestor de paquetes con repositorio Adoptium (Ubuntu/Debian, recomendado):

  # Agregar repositorio
wget -O - https://packages.adoptium.net/artifactory/api/gpg/key/public | sudo apt-key add -
echo "deb https://packages.adoptium.net/artifactory/deb $(awk -F= '/^VERSION_CODENAME/{print$2}' /etc/os-release) main" | sudo tee /etc/apt/sources.list.d/adoptium.list

# Actualizar e instalar
sudo apt-get update
sudo apt-get install temurin-21-jdk

Ubicaciones de instalación típicas

Windows:

  • Instalador: C:\Program Files\Eclipse Adoptium\jdk-21.x.x\
  • Manual: donde se extrajo (ejemplo: C:\Java\jdk-21\)

macOS:

  • Instalador: /Library/Java/JavaVirtualMachines/temurin-21.jdk/Contents/Home/
  • Manual: /Library/Java/JavaVirtualMachines/ o ~/java/

Linux:

  • Paquetes sistema: /usr/lib/jvm/temurin-21-jdk-amd64/
  • Manual sistema: /opt/jdk-21/
  • Manual usuario: ~/java/jdk-21/

Verificación post-instalación

Antes de configurar variables de entorno, verificar que los archivos fueron instalados correctamente.

Windows:

  dir "C:\Program Files\Eclipse Adoptium\jdk-21*"

macOS/Linux:

  ls /Library/Java/JavaVirtualMachines/   # macOS
ls /opt/jdk-21/                          # Linux

En ambos casos debe aparecer el directorio del JDK con la siguiente estructura:

  jdk-21.x.x/
├── bin/     # Ejecutables (java, javac, jar, etc.)
├── conf/    # Archivos de configuración
├── include/ # Headers C/C++ para JNI
├── jmods/   # Módulos en formato JMOD
├── legal/   # Licencias y avisos legales
├── lib/     # Bibliotecas runtime
└── release  # Metadata de versión

Consideraciones adicionales

Instalaciones múltiples: Es perfectamente seguro tener múltiples versiones del JDK instaladas simultáneamente. Cada versión reside en su propio directorio y la configuración de variables de entorno determina cuál está activa.

Permisos: La instalación en ubicaciones del sistema requiere privilegios administrativos. La instalación en directorios de usuario no requiere permisos especiales.

Espacio en disco: Verificar que hay al menos 500 MB a 1 GB libres. La instalación completa ocupa aproximadamente 300-400 MB.

Actualizaciones futuras: Las versiones nuevas se instalan lado a lado y no sobrescriben la versión anterior. Las versiones antiguas deben desinstalarse manualmente cuando ya no son necesarias.

La instalación exitosa del JDK es solo el primer paso. El siguiente paso crítico es configurar las variables de entorno JAVA_HOME y PATH para que el sistema pueda localizar y usar el JDK correctamente, tema que se cubre en detalle en el próximo subtema.

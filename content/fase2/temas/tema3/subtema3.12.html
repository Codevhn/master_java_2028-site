3.12: Primer programa Java para validar el entorno

Después de instalar el JDK, configurar variables de entorno, y establecer estructura de directorios, el paso final para confirmar que el entorno está completamente funcional es escribir, compilar y ejecutar un programa Java real. Este programa va más allá del simple "Hello, World" tradicional; es una herramienta de diagnóstico comprehensiva que verifica todos los aspectos del entorno de desarrollo y proporciona información detallada sobre la configuración del sistema. Este subtema guía a través de la creación de este programa de validación, explica cada componente, y demuestra cómo usarlo para confirmar que el entorno está correctamente configurado y listo para desarrollo profesional.

Objetivo del programa de validación

  1. Confirmar que javac compila código correctamente
  2. Confirmar que java ejecuta bytecode correctamente
  3. Verificar versión de Java
  4. Inspeccionar propiedades del sistema
  5. Verificar que APIs estándar son accesibles
  6. Proporcionar información de diagnóstico útil
  7. Servir como referencia de sintaxis Java básica

Preparación: Crear estructura de proyecto

Paso 1: Crear directorio del proyecto:

  # Navegar a workspace de desarrollo
cd ~/projects   # macOS/Linux
# O
cd C:\dev       # Windows

# Crear directorio para programa de validación
mkdir validacion-java
cd validacion-java

Paso 2: Crear estructura básica:

  mkdir -p src

Estructura resultante:

  validacion-java/
└── src/

Programa 1: Validación básica

Crear archivo src/ValidacionBasica.java:

  /**
 * Programa de validación básica del entorno Java.
 *
 * Este programa verifica que el JDK está instalado correctamente
 * y muestra información esencial del sistema.
 *
 * @author Tu Nombre
 * @version 1.0
 */
public class ValidacionBasica {

    public static void main(String[] args) {
        // Encabezado
        imprimirLinea('=', 60);
        System.out.println("VALIDACIÓN DE ENTORNO JAVA");
        imprimirLinea('=', 60);
        System.out.println();

        // Sección 1: Información de Java
        System.out.println("[ INFORMACIÓN DE JAVA ]");
        imprimirLinea('-', 60);
        System.out.println("Versión Java: " + System.getProperty("java.version"));
        System.out.println("Vendor: " + System.getProperty("java.vendor"));
        System.out.println("Vendor URL: " + System.getProperty("java.vendor.url"));
        System.out.println("Java Home: " + System.getProperty("java.home"));
        System.out.println("Especificación: " + System.getProperty("java.specification.version"));
        System.out.println();

        // Sección 2: Información de JVM
        System.out.println("[ INFORMACIÓN DE JVM ]");
        imprimirLinea('-', 60);
        System.out.println("Nombre JVM: " + System.getProperty("java.vm.name"));
        System.out.println("Versión JVM: " + System.getProperty("java.vm.version"));
        System.out.println("Vendor JVM: " + System.getProperty("java.vm.vendor"));
        System.out.println();

        // Sección 3: Sistema Operativo
        System.out.println("[ SISTEMA OPERATIVO ]");
        imprimirLinea('-', 60);
        System.out.println("Nombre: " + System.getProperty("os.name"));
        System.out.println("Versión: " + System.getProperty("os.version"));
        System.out.println("Arquitectura: " + System.getProperty("os.arch"));
        System.out.println();

        // Sección 4: Información de Usuario y Rutas
        System.out.println("[ USUARIO Y RUTAS ]");
        imprimirLinea('-', 60);
        System.out.println("Usuario: " + System.getProperty("user.name"));
        System.out.println("Home Usuario: " + System.getProperty("user.home"));
        System.out.println("Directorio Actual: " + System.getProperty("user.dir"));
        System.out.println("Separador Rutas: " + System.getProperty("file.separator"));
        System.out.println("Separador Path: " + System.getProperty("path.separator"));
        System.out.println();

        // Sección 5: Memoria de JVM
        System.out.println("[ MEMORIA JVM ]");
        imprimirLinea('-', 60);
        Runtime runtime = Runtime.getRuntime();
        long memoriaMaxima = runtime.maxMemory();
        long memoriaTotal = runtime.totalMemory();
        long memoriaLibre = runtime.freeMemory();
        long memoriaUsada = memoriaTotal - memoriaLibre;

        System.out.println("Memoria Máxima: " + formatearBytes(memoriaMaxima));
        System.out.println("Memoria Total: " + formatearBytes(memoriaTotal));
        System.out.println("Memoria Usada: " + formatearBytes(memoriaUsada));
        System.out.println("Memoria Libre: " + formatearBytes(memoriaLibre));
        System.out.println();

        // Sección 6: Procesadores
        System.out.println("[ PROCESADORES ]");
        imprimirLinea('-', 60);
        int procesadores = runtime.availableProcessors();
        System.out.println("Procesadores: " + procesadores + " disponibles");
        System.out.println();

        // Resultado final
        imprimirLinea('=', 60);
        System.out.println("✓ VALIDACIÓN COMPLETADA EXITOSAMENTE");
        System.out.println("  El entorno Java está correctamente configurado.");
        imprimirLinea('=', 60);
    }

    /**
     * Imprime una línea de caracteres repetidos.
     *
     * @param caracter El carácter a repetir
     * @param longitud Número de veces a repetir
     */
    private static void imprimirLinea(char caracter, int longitud) {
        for (int i = 0; i < longitud; i++) {
            System.out.print(caracter);
        }
        System.out.println();
    }

    /**
     * Formatea bytes a formato legible (KB, MB, GB).
     *
     * @param bytes Número de bytes
     * @return String formateado con unidad apropiada
     */
    private static String formatearBytes(long bytes) {
        if (bytes < 1024) {
            return bytes + " B";
        } else if (bytes < 1024 * 1024) {
            return String.format("%.2f KB", bytes / 1024.0);
        } else if (bytes < 1024 * 1024 * 1024) {
            return String.format("%.2f MB", bytes / (1024.0 * 1024.0));
        } else {
            return String.format("%.2f GB", bytes / (1024.0 * 1024.0 * 1024.0));
        }
    }
}

Compilación del programa

Paso 1: Navegar al directorio del proyecto:

  cd ~/projects/validacion-java

Paso 2: Compilar el código fuente:

  javac src/ValidacionBasica.java

Sin salida significa éxito. Verificar que se creó src/ValidacionBasica.class:

  ls -l src/   # macOS/Linux
dir src\     # Windows

Debe mostrar ambos archivos: ValidacionBasica.java (código fuente) y ValidacionBasica.class (bytecode compilado).

Errores comunes de compilación:

  • "error: cannot find symbol": verificar que el código fue copiado exactamente, respetando mayúsculas/minúsculas en nombres de clases y métodos.
  • "javac: command not found": PATH no está configurado correctamente — revisar subtema 3.6.

Ejecución del programa

  # Desde directorio raíz del proyecto (validacion-java/)
java -cp src ValidacionBasica

-cp src establece el classpath al directorio src donde está el .class. ValidacionBasica es el nombre de la clase (sin extensión).

Análisis de la salida

Versión Java correcta: debe coincidir con la versión instalada. Si muestra versión diferente, otra instalación de Java tiene precedencia en PATH.

Vendor correcto: Eclipse Adoptium, Amazon Corretto, Azul, etc. Confirma qué distribución está activa.

Java Home correcto: debe apuntar a la instalación del JDK y coincidir con JAVA_HOME.

Arquitectura apropiada: aarch64 para Apple Silicon (M1/M2/M3); x86_64 o amd64 para Intel/AMD de 64-bit. Si muestra 32-bit, la instalación es incorrecta.

Memoria disponible razonable: Memoria Máxima debe ser sustancial (varios GB). Si es muy baja (<512 MB), puede indicar límites de JVM no configurados apropiadamente.

Programa 2: Validación avanzada con tests de funcionalidad

Crear archivo src/ValidacionAvanzada.java:

  import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.nio.file.Paths;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

/**
 * Programa de validación avanzada que testea funcionalidad de APIs Java.
 * Verifica que APIs estándar están accesibles y funcionan correctamente.
 */
public class ValidacionAvanzada {

    private static int testsEjecutados = 0;
    private static int testsExitosos = 0;
    private static int testsFallidos = 0;

    public static void main(String[] args) {
        imprimirEncabezado();
        testStrings();
        testColecciones();
        testFechasYTiempo();
        testSistemaArchivos();
        testOperacionesMatematicas();
        imprimirResumen();
    }

    private static void imprimirEncabezado() {
        System.out.println("╔════════════════════════════════════════════════════════════╗");
        System.out.println("║         VALIDACIÓN AVANZADA DE ENTORNO JAVA               ║");
        System.out.println("╚════════════════════════════════════════════════════════════╝");
        System.out.println();
    }

    private static void testStrings() {
        System.out.println("[ TEST: Operaciones con Strings ]");
        System.out.println("----------------------------------------------------------");

        ejecutarTest("Concatenación de strings", () -> {
            String saludo = "Hola";
            String nombre = "Java";
            String resultado = saludo + " " + nombre;
            return resultado.equals("Hola Java");
        });

        ejecutarTest("Métodos de String", () -> {
            String texto = " Desarrollo Java ";
            return texto.trim().toUpperCase().contains("JAVA");
        });

        ejecutarTest("StringBuilder", () -> {
            StringBuilder sb = new StringBuilder();
            sb.append("Java").append(" ").append("21");
            return sb.toString().equals("Java 21");
        });

        System.out.println();
    }

    private static void testColecciones() {
        System.out.println("[ TEST: Colecciones ]");
        System.out.println("----------------------------------------------------------");

        ejecutarTest("ArrayList", () -> {
            List<String> lista = new ArrayList<>();
            lista.add("Java");
            lista.add("Python");
            lista.add("JavaScript");
            return lista.size() == 3 && lista.contains("Java");
        });

        ejecutarTest("HashMap", () -> {
            Map<String, Integer> mapa = new HashMap<>();
            mapa.put("Java", 21);
            mapa.put("Python", 3);
            return mapa.get("Java") == 21;
        });

        System.out.println();
    }

    private static void testFechasYTiempo() {
        System.out.println("[ TEST: Fechas y Tiempo (java.time API) ]");
        System.out.println("----------------------------------------------------------");

        ejecutarTest("LocalDateTime", () -> {
            LocalDateTime ahora = LocalDateTime.now();
            return ahora != null && ahora.getYear() >= 2024;
        });

        ejecutarTest("DateTimeFormatter", () -> {
            LocalDateTime fecha = LocalDateTime.of(2024, 1, 15, 10, 30);
            DateTimeFormatter formato = DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm");
            String fechaFormateada = fecha.format(formato);
            return fechaFormateada.equals("15/01/2024 10:30");
        });

        System.out.println();
    }

    private static void testSistemaArchivos() {
        System.out.println("[ TEST: Sistema de Archivos (NIO.2) ]");
        System.out.println("----------------------------------------------------------");

        ejecutarTest("Paths y Files", () -> {
            try {
                Path rutaTemporal = Paths.get(System.getProperty("java.io.tmpdir"));
                return Files.exists(rutaTemporal) && Files.isDirectory(rutaTemporal);
            } catch (Exception e) {
                return false;
            }
        });

        ejecutarTest("Crear archivo temporal", () -> {
            try {
                Path archivoTemp = Files.createTempFile("test-java-", ".tmp");
                boolean existe = Files.exists(archivoTemp);
                Files.deleteIfExists(archivoTemp);
                return existe;
            } catch (IOException e) {
                return false;
            }
        });

        System.out.println();
    }

    private static void testOperacionesMatematicas() {
        System.out.println("[ TEST: Operaciones Matemáticas ]");
        System.out.println("----------------------------------------------------------");

        ejecutarTest("Operaciones básicas", () -> {
            int suma = 10 + 5;
            int resta = 10 - 5;
            int multiplicacion = 10 * 5;
            double division = 10.0 / 4.0;
            return suma == 15 && resta == 5 && multiplicacion == 50 && division == 2.5;
        });

        ejecutarTest("Clase Math", () -> {
            double raiz = Math.sqrt(16);
            double potencia = Math.pow(2, 3);
            int maximo = Math.max(10, 20);
            return raiz == 4.0 && potencia == 8.0 && maximo == 20;
        });

        System.out.println();
    }

    private static void ejecutarTest(String nombreTest, TestCallable test) {
        testsEjecutados++;
        try {
            boolean resultado = test.call();
            if (resultado) {
                testsExitosos++;
                System.out.println("  ✓ " + nombreTest + " - ÉXITO");
            } else {
                testsFallidos++;
                System.out.println("  ✗ " + nombreTest + " - FALLO");
            }
        } catch (Exception e) {
            testsFallidos++;
            System.out.println("  ✗ " + nombreTest + " - ERROR: " + e.getMessage());
        }
    }

    private static void imprimirResumen() {
        System.out.println("╔════════════════════════════════════════════════════════════╗");
        System.out.println("║                   RESUMEN DE TESTS                        ║");
        System.out.println("╠════════════════════════════════════════════════════════════╣");
        System.out.println(String.format("║  Tests Ejecutados: %-39d║", testsEjecutados));
        System.out.println(String.format("║  Tests Exitosos:   %-39d║", testsExitosos));
        System.out.println(String.format("║  Tests Fallidos:   %-39d║", testsFallidos));
        System.out.println("╠════════════════════════════════════════════════════════════╣");

        if (testsFallidos == 0) {
            System.out.println("║  ✓ TODOS LOS TESTS PASARON                                ║");
            System.out.println("║    El entorno Java está completamente funcional.          ║");
        } else {
            System.out.println("║  ⚠ ALGUNOS TESTS FALLARON                                ║");
            System.out.println("║    Revisar configuración del entorno.                    ║");
        }

        System.out.println("╚════════════════════════════════════════════════════════════╝");
    }

    @FunctionalInterface
    interface TestCallable {
        boolean call() throws Exception;
    }
}

Qué se ha validado

Compilador funcionando: javac está instalado y accesible, compila código fuente correctamente, y genera bytecode válido.

Runtime funcionando: java está instalado y accesible, ejecuta bytecode correctamente, y la JVM inicia apropiadamente.

APIs estándar accesibles: System para propiedades del sistema, String y StringBuilder, colecciones (ArrayList, HashMap), API de tiempo (java.time), API de archivos (java.nio.file), y Math.

Funcionalidad básica verificada: creación de objetos, llamadas a métodos, operaciones aritméticas, manejo de excepciones, y entrada/salida básica.

Siguientes pasos después de validación exitosa

  1. Instalar IDE (IntelliJ IDEA, Eclipse, VS Code con extensiones Java)
  2. Configurar sistema de construcción (Maven o Gradle)
  3. Comenzar desarrollo de proyectos reales
  4. Explorar frameworks (Spring Boot, Jakarta EE, etc.)

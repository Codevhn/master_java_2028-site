3.2: Requisitos previos del sistema operativo

Antes de instalar el JDK, es fundamental verificar que la máquina cumple con los requisitos mínimos de hardware y software necesarios para ejecutar Java efectivamente.

Requisitos de hardware

Memoria RAM:

  • Mínimo: 2 GB
  • Recomendado: 4 GB para desarrollo básico
  • Ideal: 8 GB o más (IDEs, múltiples aplicaciones, proyectos grandes)

La memoria es crítica porque el proceso de compilación, ejecución de aplicaciones Java, y los IDEs modernos (IntelliJ IDEA, Eclipse) consumen recursos significativos.

Espacio en disco:

  • Instalación JDK: 300 MB - 1 GB
  • Entorno completo recomendado: mínimo 10 GB libres
  • Proyectos y código fuente
  • Dependencias Maven/Gradle (pueden acumular varios GB)
  • IDE (1-3 GB)
  • Archivos de construcción generados

Procesador:

  • Cualquier procesador moderno x86-64 (64-bit) de los últimos 10 años
  • Procesadores multi-core benefician significativamente la compilación de proyectos grandes (paralelización de builds)

Sistemas operativos soportados

Windows:

  • Versiones soportadas: Windows 10, Windows 11
  • Versiones anteriores: Windows 8.1 (según versión JDK)
  • Solo versiones 64-bit son soportadas por JDKs modernos

macOS:

  • Versiones: macOS 10.13 (High Sierra) y posteriores
  • Intel x86-64: builds específicos
  • Apple Silicon (ARM64/M1/M2/M3): builds específicos
  • Descargar el build correcto para la arquitectura del procesador

Linux:

  • Distribuciones soportadas: Ubuntu, Debian, Fedora, RHEL, CentOS, otras mayores
  • Versiones: Distribuciones lanzadas en los últimos 3-5 años generalmente soportadas
  • glibc en versión apropiada (Debian/Ubuntu)
  • Bibliotecas equivalentes (sistemas basados en RPM)

Permisos y privilegios

Instalación con privilegios administrativos:

  • Windows: Ejecutar instalador como Administrador
  • macOS/Linux: Privilegios sudo o root para instalaciones en ubicaciones del sistema

Instalación sin privilegios administrativos:

  • Descomprimir manualmente archivo tar.gz o zip en directorio de usuario
  • No requiere permisos especiales, pero configuración manual de variables de entorno necesaria

Entornos corporativos:

  • Coordinar con IT para instalación o permisos apropiados
  • Algunas organizaciones proporcionan imágenes pre-configuradas con JDK

Software adicional requerido o recomendado

Linux:

  • Bibliotecas X11 (para aplicaciones Java con GUI)
  • gcc y herramientas de desarrollo (si se usa JNI o compilación de código nativo)

macOS:

  • Xcode Command Line Tools (beneficioso, especialmente si se usa Homebrew)

Todos los sistemas:

  • Navegador web moderno (documentación, descargas, herramientas web-based)

Verificación de requisitos previos

Verificar versión del sistema operativo — Windows:

  winver
systeminfo

Verificar versión del sistema operativo — macOS:

  sw_vers

Verificar versión del sistema operativo — Linux:

  lsb_release -a
# o
cat /etc/os-release

Verificar memoria RAM:

  • Windows: Task Manager → Rendimiento
  • macOS: Activity Monitor → Memoria
  • Linux: free -h

Verificar espacio en disco:

  • Windows: Explorador de Archivos → Propiedades de unidad
  • macOS/Linux: df -h

Cumplir estos requisitos asegura instalación exitosa del JDK y entorno de desarrollo funcional.
